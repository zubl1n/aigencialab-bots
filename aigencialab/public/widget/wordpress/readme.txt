AIgenciaLab Chatbot es la solución definitiva para automatizar la atención al cliente y captura de leads en tu web.

== Description ==

Con este plugin podrás integrar el chatbot de IA de AIgenciaLab en SEGUNDOS. Solo necesitas tu API ID.

== Installation ==

1. Sube la carpeta del plugin al directorio `/wp-content/plugins/`.
2. Activa el plugin en el menú 'Plugins'.
3. Ve a 'Ajustes -> AIgenciaLab Bot' e ingresa tu API Key.
